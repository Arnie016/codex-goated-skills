# Icon Bar Skill Categories

This page groups the newer macOS Icon Bars skills into GitHub-friendly buckets so the catalog is easier to scan than a flat table.

Audit date: 2026-05-06.

## Audit Result

- The 13 listed skills all have `SKILL.md`, `manifest.json`, icons, default prompts, and pack coverage.
- `clipboard-studio` was already surfaced in the catalog and pack files, but was missing `skills/clipboard-studio/manifest.json`; this page includes it and the manifest has been added.
- These categories are browse groupings. Pack membership remains the install source of truth.
