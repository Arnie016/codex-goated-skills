# Focus Runway SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Open the next work block with less friction.
- A focus launcher that stays quiet until you need it.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Launch: Pick a block, open the right surfaces, and reduce the pre-work setup loop.
- Guardrails: Keep it light and avoid turning focus into another massive dashboard.
- Tone: The interface should feel like a small helper instead of a command center.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
