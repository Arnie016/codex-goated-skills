# Meeting Link Bridge SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Turn the current meeting tab into one clean note or join action.
- A quick join-link relay for Teams-heavy meeting handoffs.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Provider status: Show the detected provider, source lane, and meeting code in one compact top strip.
- Handoff formats: Keep note, email, and markdown outputs one tap away instead of rebuilding the join note each time.
- Join lane: Surface one open action and one copy action so the pre-meeting switch stays short and calm.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
