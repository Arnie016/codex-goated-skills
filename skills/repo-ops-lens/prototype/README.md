# Repo Ops Lens SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Turn a repo into a fast operating brief.
- A menu-bar repo lens for quick audits and planning.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- What it scans: The app can summarize a repo, surface high-risk areas, and show the next safe command or fix to run.
- Why it matters: It cuts the time spent switching between browser, terminal, and notes when you only need the gist.
- UI tone: Use a narrow, readable panel with strong hierarchy and one primary recommendation.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
