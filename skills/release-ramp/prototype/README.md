# Release Ramp SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- A calmer path from ready-to-ship to live.
- A launch lane for tiny teams and solo builders.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Launch surface: The app shows the current release step, what is left, and the next clean move.
- Why it helps: It reduces the friction of remembering release steps when the room is already busy.
- Design direction: Use one bold action, two supporting checks, and a restrained progress strip.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
