# Server Modes

Use this to choose the right Minecraft server lane before touching jars or plugins.

## Vanilla

Best when:

- the user wants the clean official server
- there are no plugins or mod requirements
- simplicity matters more than admin tooling

Tradeoffs:

- fewer admin features
- no Paper plugin ecosystem
- weaker performance story for busy multiplayer worlds

## Paper

Best when:

- the server is for practical multiplayer play
- plugins are needed
- the user wants better performance and admin tooling
- compatibility plugins or Bedrock bridging may be added later

Tradeoffs:

- not a pure Vanilla binary
- plugin compatibility still depends on server version

## Fabric

Best when:

- the server needs Fabric mods
- the user is intentionally building a modded stack
- plugin-only solutions are not enough

Tradeoffs:

- different ecosystem from Paper
- more moving parts for pack compatibility

## Good Default

If the user wants a stable local multiplayer server with practical upgrades, default to Paper.
