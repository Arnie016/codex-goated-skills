# Plugin Packs

Use focused bundles instead of random plugin piles.

## Java Compatibility Pack

- ViaVersion
- ViaBackwards
- ViaRewind

Use when older Java clients need to join a newer Paper server.

Important:

- this does not make the world truly old-version native
- best experience is still near the server's actual version

## Bedrock Bridge Pack

- Geyser
- Floodgate

Use when Bedrock players need to join a Java-hosted server.

Important:

- Bedrock support is a bridge layer, not a server-mode replacement
- always verify plugin versions against the server version

## Admin Pack

Examples:

- LuckPerms
- CoreProtect
- EssentialsX
- WorldEdit

Use when the server needs moderation, permissions, rollback, or staff tools.

## Survival QoL Pack

Examples:

- homes
- claims
- teleport or spawn tools
- anti-grief basics

Use when a small friend-group server wants convenience without heavy modding.

## Avoid

- mixing too many overlapping admin plugins
- upgrading many plugins at once without a backup
- assuming old plugins work on the newest Paper build
