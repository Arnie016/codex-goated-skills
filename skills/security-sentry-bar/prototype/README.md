# Security Sentry Bar SwiftUI prototype

This folder sketches the native macOS menu-bar version of the SwiftBar plugin.

## What it sketches

- A compact host-security status surface with green, yellow, and red state.
- Local evidence sections for connections, listening ports, LaunchAgents, processes, and recent files.
- Explicit scan actions instead of hidden persistence.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`
