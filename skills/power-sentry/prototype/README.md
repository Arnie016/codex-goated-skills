# Power Sentry SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Know your battery state without opening a panel full of clutter.
- A compact power monitor with trend context and a clean tone.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- At a glance: Show charge, power source, and energy mode first so the user can check the state in one glance.
- Deeper read: Add trend context only when it helps a real decision like charging or low power mode.
- Design: Keep the styling monochrome and premium so it feels like a first-party utility.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
