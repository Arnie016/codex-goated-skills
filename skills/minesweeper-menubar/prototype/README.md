# Minesweeper Menu Bar SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- A Swift menu-bar icon app prototype for playing Minesweeper.
- A compact popover board with reveal and flag interactions.
- A direct template for turning a classic puzzle into a native Mac top-bar micro game.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillGameModel.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The prototype keeps the game local, compact, and menu-bar first.
