# Minefield Menu Bar SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- A genuinely playable Minesweeper-style micro game for the macOS menu bar.
- A compact board with reveal and flag flow, quick reset, and light progression stats.
- A strong shell for building more top-bar arcade concepts once the interaction feels right.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillGameModel.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Board loop: Fit a full game round into a compact menu-bar popover.
- Replay feel: Keep reset and rematch faster than setup.
- Expansion lane: Reuse the shell for more polished micro games later.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The prototype keeps the game fully local and menu-bar first.
