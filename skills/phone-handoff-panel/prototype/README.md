# Phone Handoff Panel SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Keep the phone handoff close without leaving the Mac.
- A connection panel that makes device switching feel deliberate.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Device loop: Surface the phone actions that matter now instead of every possible option.
- Behavior: Lead with open, locate, and continue actions because those are the ones people repeat.
- Mac feel: The UI should stay calm, compact, and native so it belongs in the menu bar.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
