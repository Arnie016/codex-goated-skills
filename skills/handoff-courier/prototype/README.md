# Handoff Courier SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Move work between apps without losing context.
- A menu-bar courier for file, snippet, and export handoffs.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- What it does: Handoff Courier gives you a small queue for files, clipboard payloads, and exports so transfers feel like one deliberate step instead of a chain of window switches.
- Why it helps: It is meant for the exact moment where you are bouncing between Codex, Finder, Telegram, browser tabs, and notes and just want a clean drop path.
- SwiftUI shape: Use a menu-bar popover, a compact transfer card, and a short action row with calm labels so the app behaves like a real Mac utility.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
