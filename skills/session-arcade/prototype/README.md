# Session Arcade SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Start the next session without bouncing through extra screens.
- A compact gaming helper for launch-night flow.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Session flow: Surface the next game, the current launcher state, and the fastest path to play.
- Why it works: It saves the small but annoying setup steps that happen before you actually play.
- Mood: Keep it playful, polished, and short enough to fit in the top bar.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
