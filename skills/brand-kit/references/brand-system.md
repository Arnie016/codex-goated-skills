# Brand System

Use this structure when producing a small but useful brand package.

## Positioning

- product name
- one-line value proposition
- audience
- tone

## Visual Direction

- primary color
- accent color
- background or neutral
- typography direction

## Logo Direction

Include:

- symbol concept
- shape language
- what the icon should imply
- what to avoid

## Launch Metadata

- GitHub or page title
- meta description
- hero headline
- hero subhead
- social preview text

## Good Output Traits

- easy to paste into a README or launch page
- easy to hand off to a designer or image generator
- consistent across logo, copy, and colors
