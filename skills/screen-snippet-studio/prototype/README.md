# Screen Snippet Studio SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Capture the moment and turn it into something useful.
- A tiny screen-to-prompt utility that stays out of the way.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Capture flow: Take a quick shot, add a short note, and route it into the right workspace without opening a full editor.
- Signal over noise: The UI should feel more like a quiet staging area than a complicated media tool.
- Output: Use the result as a prompt, a bug note, or a handoff artifact.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
