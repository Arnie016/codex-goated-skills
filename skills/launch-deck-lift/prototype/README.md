# Launch Deck Lift SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Turn a rough thought into a cleaner presentation shape.
- A deck starter for fast launches and meetings.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Flow: Start with the story, then map out the slides that support it.
- Why it helps: It removes the blank-page moment when the deck needs to happen quickly.
- UI: Use a structured outline and a simple preview surface instead of a giant canvas.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
