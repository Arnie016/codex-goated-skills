# Story Arc Board SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Catch recurring hooks before they vanish into tab sprawl.
- A menu-bar board for notes, snippets, and repeated audience beats.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Capture lane: Collect phrases, screenshots, and quick notes from the apps you already have open.
- Why it helps: It turns scattered narrative scraps into one visible queue before context disappears.
- Menu-bar shape: Use a compact popover with pinned beats, a short intake list, and one clean action to promote the next arc.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
