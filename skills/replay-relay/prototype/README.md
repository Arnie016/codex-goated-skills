# Replay Relay SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Turn a saved match moment into a clean share handoff in one stop.
- A compact clip-and-screenshot relay for game nights, posts, and friend chats.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Capture intake: Show the latest screenshot or a dropped clip as one staging card with preview, title, size, and destination badge already resolved.
- Share pass: Lead with rename, caption, copy path, and reveal actions so the user can get from capture to Discord, group chat, or notes without Finder detours.
- Menu-bar tone: Keep the popover playful but disciplined, like a Mac utility that understands gaming rituals without turning into a noisy dashboard.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
