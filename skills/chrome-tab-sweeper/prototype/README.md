# Chrome Tab Sweeper SwiftUI prototype

This folder contains a generated macOS menu-bar starter for the skill package.

## What it sketches

- A top-bar browser control layer for people with too many Chrome tabs open.
- A domain summary so tab chaos becomes readable fast.
- A selected cleanup batch that closes only after review.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`
