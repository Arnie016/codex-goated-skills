# Doc Drop Bridge SwiftUI prototype

This folder is the generated macOS menu-bar starter that pairs with the skill package.

## What it is

- Package a working note into a cleaner handoff.
- A document bridge that stays small and intentional.

## Included files

- `SkillMenuBarApp.swift`
- `SkillMenuBarView.swift`
- `SkillDetailView.swift`
- `SkillTheme.swift`

## Prototype sections

- Input: Drop in a note, outline, or rough draft and the bridge prepares it for sharing.
- Output: Choose a compact export path that keeps the result easy to move into another app.
- Presentation: The interface should read like an Apple utility with one clear action at a time.

## Notes

- Drop these files into a macOS SwiftUI target if you want to turn the sketch into a real app.
- The skill market will surface the package metadata, while this folder keeps the SwiftUI shape close by.
